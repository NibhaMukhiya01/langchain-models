# LangChain Models Reference & Practice

Welcome to my personal repository for exploring and practicing with LangChain model components! 

This repository is a fork of the original [campusx-official/langchain-models](https://github.com/campusx-official/langchain-models) project.
I have saved it here as a reference guide and sandbox to build out, test, and implement different LangChain concepts as I learn.



## 📂 Repository Structure

The project is organized into the following core components of the LangChain model ecosystem:

*   **`1.LLMs`** – Exploring large language models that take a text string as input and return a text string.
*   **`2.ChatModels`** – Working with chat models backed by a language model, designed for multi-turn
*    conversational interfaces (taking structured messages as input and outputting a message).
*   **`3.EmbeddingModels`** – Implementing text embeddings to convert words and phrases into numerical
*    vectors for semantic search and retrieval.
